# Online-Shopping-Cart
The Shopping Cart System is a simple console-based Java application that allows users to view store items, add items to a cart, view cart items, and calculate the total bill. This project demonstrates the use of Java Collections (HashMap), OOP concepts, and menu-driven programming.
